# HackUPC2022-Mundimonto

Hackathon project from HackUPC 2022 — data analysis on a motorcycle dataset provided by Mundimoto.

## The Problem

- Mundimoto provided a real-world dataset of motorcycle listings and sales data as a hackathon challenge.
- The challenge required extracting insights and building something useful from the data within a 36-hour event.

## The Approach

**Inputs:** `motorcycles.csv` — a dataset of motorcycle listings including attributes such as make, model, price, year, and condition. Raw data files are in `mundimoto_rawdata/`.

**Processing:** Exploratory data analysis (EDA) and modelling performed in `mundimoto.ipynb`. Reference materials in `ref/` informed the approach.

**Outputs:** Insights and analysis presented during the hackathon.

## Value Delivered

- Demonstrated ability to rapidly explore and analyse an unfamiliar dataset under time pressure.
- Produced actionable findings within the hackathon timeframe.
